Logic tư duy lập trình:

  • State: Sử dụng biến counter để theo dõi ảnh hiện tại.
  • DOM: Truy xuất chiều rộng của ảnh (`clientWidth`) để biết cần dịch chuyển bao nhiêu pixel.
  • Transition: Dùng `transform: translateX()` kết hợp CSS `transition` để tạo hiệu ứng trượt.
  • Xử lý biên: Khi bấm Next ở ảnh cuối cùng, code sẽ reset `counter` về 0. Ngược lại với nút Prev.
  • Tự động: Dùng `setInterval` gọi hàm `nextSlide` mỗi 3s. Khi người dùng click nút, clear interval và set lại để tránh xung đột.
